# Rare Ores Datapack

A hardcore difficulty datapack that makes survival significantly harder by drastically reducing resource availability.

## Features

### Ore Generation Changes

All overworld and nether ores spawn **100x less frequently** than vanilla. This affects:

- **Coal, Iron, Copper, Gold** - All variants (upper, middle, lower, large veins)
- **Diamonds** - All variants (regular, medium, large, buried)
- **Redstone, Lapis Lazuli, Emeralds**
- **Nether ores** - Quartz, Nether Gold, Ancient Debris (both large and small)

Vein sizes and height distributions remain vanilla, but spawn frequency is reduced by 99%.

### Mob Loot Nerfs

- **Drowned**: Copper ingots removed entirely from loot table (only drops rotten flesh with 2.5% base chance + looting)
- **Iron Golems**: Iron ingots removed (only drops 0-2 poppies)
- **Piglins & Zombified Piglins**: Gold ingots/nuggets removed from drops

### Village Removal

Villages are completely disabled and will not generate naturally in the world.

### Mending Nerf

Any villager that offers mending enchanted books for trade is instantly killed. This makes mending practically unobtainable through villager trading.

## Gameplay Impact

This pack forces players into extreme resource scarcity:
- Mining becomes 100x less rewarding
- Mob farms don't yield metals
- Villages don't spawn
- Mending is practically unobtainable

Every resource must be carefully managed. Suitable for players seeking an ultra-hardcore survival challenge.

## Installation

1. Download the datapack
2. Place it in your world's `datapacks` folder
3. Run `/reload` in-game or restart the server

## Compatibility

- **Pack Format**: 75
- **Supported Formats**: 75, 81
- Compatible with Minecraft 1.21+
